## Keyword Implementation Tips

### Keyword Density Target
- Aim for 0.5-3% keyword density
- Spread keywords naturally throughout content
- Use in titles, headings, meta descriptions
- Include in image alt text

### Where to Use Keywords

1. **Page Title** (50-60 chars)
   - Include primary keyword
   - Include brand name
   
2. **Meta Description** (150-160 chars)
   - Include primary keyword
   - Clear call-to-action
   
3. **H1 Headings**
   - Include main keyword
   - Make compelling for users
   
4. **Body Content**
   - Natural integration
   - Related keywords/synonyms
   
5. **Image Alt Text**
   - Descriptive with keyword where relevant
   - Never keyword stuff
